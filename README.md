# Новое приложение
Новый андроид клиент для имиджборд (бывший Dashchan и 2ChBrowser)

# Схожие приложения
Dashchan Overchan 2ChBrowser

# Поддержка
Андроид 4.1 - 11.0

# APK для установки
https://github.com/TotalDay/Dashchan-New/releases/download/V3.1.3/default.apk
